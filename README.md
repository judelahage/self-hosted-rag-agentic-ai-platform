# Self-Hosted RAG and Agentic AI Platform

A full-stack AI application for private document search and question answering, powered by local LLMs, Elasticsearch vector search, and multi-step agentic workflows.

## Status

Work in progress — Summer 2026

## Tech Stack

- **Frontend:** Next.js + TypeScript
- **Backend:** FastAPI + Python
- **LLM Runtime:** Ollama (local)
- **Vector Database:** Elasticsearch
- **RAG Framework:** Haystack / LangChain
- **Agent Layer:** LangGraph / CrewAI
- **Containerization:** Docker
